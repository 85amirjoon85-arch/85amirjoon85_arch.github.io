درس ۳: شرط if

با شرط‌ها می‌توانیم تصمیم‌گیری کنیم. 


#include 
using namespace std;

int main() {
    int x;
    cin >> x;

    if(x > 0)
        cout << "مثبت";
    else
        cout << "منفی یا صفر";

    return 0;
}

اگر عدد مثبت باشد پیام «مثبت» چاپ می‌شود.
